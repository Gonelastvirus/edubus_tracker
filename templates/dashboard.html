🚀 Quick Start Guide

  1. Create Admin Account: Use the Admin Panel to create your first administrator account
  2. Add Buses: Register school buses with driver information
  3. Create Stations: Define bus stops with GPS coordinates and route order
  4. Add Students: Register students and assign them to buses and stations
  5. Test Tracking: Use the Bus Simulator to send GPS updates and test the system
  6. Student Access: Students can now track their bus in real-time
